## 🥇 Top10 详评

### 1. 绿的谐波（688017）— 90 ｜ 谐波减速器
- **尖端度**：中国天花板、全球第二梯队最强（对标日本 Harmonic Drive）。
- **卡点判据**：谐波减速器是人形机器人关节核心，材料/齿形/精度/寿命四重壁垒、精度一致性**难以扩产**，全球寡头格局。
- **估值/成长**：PE549 高但远期人形机器人量产（2026→10 万台、2027→50 万台）空间极大，**不按 PE 锚定，用谐波出货量验证**。
- **风险/证伪**：人形量产节奏低于预期；来福谐波分流份额。
- **维度分**：卡点31 / 倒挂17 / 成长19 / 周期11 / 拐点7 / 流动5。
- **唯一兼具「赛道天花板 + 中小市值(696亿) + 大成长」的标的，名副其实第一。**

### 2. 天孚通信（300394）— 85 ｜ CPO 硅光引擎
- **尖端度**：最尖端——**直供英伟达 1.6T 硅光引擎**，掌握 FAU 封装与 ERS 激光光源，CPO 无源生态卡位最纯。
- **卡点判据**：被全球顶级客户（英伟达）选中即尖端认证；CPO 量产临近（英伟达 2026 Q4）。
- **降分原因**：3539 亿市值、PB59，市值-壁垒倒挂仅 9/20。
- **维度分**：卡点35 / 倒挂9 / 成长18 / 周期12 / 拐点8 / 流动3。

### 3. 东山精密（002384）— 85 ｜ 光芯片 EML（索尔思）
- **尖端度**：经索尔思（Source Photonics）成为**国内唯一 200G EML 大规模量产**厂商，自供率 99%、已进英伟达/Meta 供应链、领先行业 1-2 年；**中际旭创 200G EML 主供即东山**。
- **卡点判据**：200G EML 是 1.6T 光模块核心光源，外延良率难扩产、全球紧缺。
- **降分原因**：4762 亿大市值 + 业务杂（PCB+光模块+光芯片），纯度低、倒挂仅 9/20。
- **维度分**：卡点34 / 倒挂9 / 成长17 / 周期12 / 拐点8 / 流动5。

### 4. 安集科技（688019）— 84 ｜ CMP 抛光液
- **尖端度**：全球前三（约 10% 份额）、全品类+自研磨料。
- **卡点判据**：CMP 抛光液高频耗材、认证周期长难替换，材料卡点；**赛道天花板 + 540 亿中小盘**，倒挂 17/20。
- **估值**：PE51 在卡点票里相对低。
- **维度分**：卡点30 / 倒挂17 / 成长16 / 周期11 / 拐点7 / 流动3。

### 5. 鼎龙股份（300054）— 84 ｜ CMP 抛光垫
- **尖端度**：全球少数、**国内唯一掌握全流程**（陶氏之外），核心原料微球自产。
- **卡点判据**：抛光垫国产化此前空白，配方/微球/制程一体化壁垒极高、难扩产。
- **风险**：打印耗材旧业务拖累，需拆半导体材料看增速。
- **维度分**：卡点30 / 倒挂15 / 成长16 / 周期11 / 拐点7 / 流动5。

### 6. 江丰电子（300666）— 79 ｜ 高纯溅射靶材
- **尖端度**：全球前列——高纯金属溅射靶材进入台积电/三星等先进制程供应链。
- **卡点判据**：靶材纯度（5N5+）与晶粒控制壁垒高、认证久，半导体制造耗材卡点。
- **维度分**：卡点28 / 倒挂15 / 成长15 / 周期10 / 拐点6 / 流动5。

### 7. 中际旭创（300308）— 76 ｜ 光模块整机
- **尖端度**：**全球第一**（技术+份额双第一），800G/1.6T 出货确定性最强。
- **降分原因**：1.4 万亿巨头，市值-壁垒倒挂仅 5/20——卡点真实性满分(35)，但已远非"被忽视小盘"。
- **维度分**：卡点35 / 倒挂5 / 成长14 / 周期12 / 拐点8 / 流动2。

### 8. 新易盛（300502）— 76 ｜ 光模块整机
- **尖端度**：全球第一梯队，1.6T 放量；PE57 在光模块龙头里最低。
- **降分原因**：7679 亿大市值，倒挂仅 5/20。
- **维度分**：卡点33 / 倒挂5 / 成长15 / 周期12 / 拐点8 / 流动3。

### 9. 中微公司（688012）— 76 ｜ 介质刻蚀设备
- **尖端度**：全球先进——介质刻蚀已通过 5nm 验证，国产设备里最接近国际一线（对标 Lam/TEL）。
- **卡点判据**：刻蚀是晶圆制造核心工艺设备，介质刻蚀技术壁垒高。
- **降分原因**：3045 亿市值，倒挂 9/20。
- **维度分**：卡点31 / 倒挂9 / 成长16 / 周期11 / 拐点7 / 流动2。

### 10. 天岳先进（688234）— 74 ｜ SiC 导电衬底
- **尖端度**：全球前三（Wolfspeed/Coherent 之后）。
- **卡点判据**：SiC 导电型衬底长晶+加工壁垒高、玩家极少。
- **降分原因**：⚠️ **边界外缘**——SiC 主要用于车规/功率，与 AI 算力关联弱；当前亏损+价格战。
- **维度分**：卡点27 / 倒挂15 / 成长13 / 周期10 / 拐点5 / 流动4。

## ⚠️ Serenity 体系内在张力（重要）

1. **"卡脖子要最尖端" 与 "看重小市值" 本质冲突**：能做到全球技术最尖端的公司，通常已经做大了（天孚 3539 亿 / 东山 4762 亿 / 中际旭创 1.4 万亿 / 新易盛 7679 亿）。
2. **A股 真·技术天花板级卡脖子稀缺**：全市场够格的就约 10 只，硬凑不出更多。
3. **甜点稀少**：同时满足"赛道天花板 + 中小市值"的仅 **绿的谐波 / 安集 / 鼎龙 / 江丰** 四只——这才是 Serenity 真正的核心仓。
4. **国产替代 ≠ 卡脖子**：A股 大量"国产化率低"的标的（光芯片二梯队、丝杠、力传感器、涂胶显影、离子注入）是政策窗口下的追赶者，全球技术非天花板，均不入卡脖子池。
